# R45 Music
Web Based Music Player that can play music from Youtube and Joox  
Menggunakan NodeJs dan beberapa framework untuk memutar musik dari youtube dan joox

## Installation
Use the package manager NPM to install R45 Music.

```javascript
npm install
```

## Usage

```javascript
npm start
```
